# Editor 页面规格

## 页面目标

让用户围绕章节树组织教材，并把外部生成的 Markdown 快速粘贴、校对和预览。

## 页面区域与组件层级

- `EditorHeader`：面包屑、可修改的文档信息、保存、预览、进入阅读。
- `ChapterSidebar`
  - 新建章节/小节、导入 Markdown 章节。
  - `ChapterTree`：展开、选中、删除、上移、下移。
- 内容工作区
  - `ContentModeTabs`：正文与素材分离。
  - `MarkdownToolbar`：插入标题、强调、列表、引用、链接、代码、表格、公式和分割线，整理格式，上传和管理图片。
  - `MarkdownEditor` / `MarkdownPreview`。
  - `SaveStatus`：已保存、保存中、有未保存更改。

## 用户交互

- 切换章节加载对应内容；编辑正文或素材触发保存状态变化。
- 手动保存立即写入 SQLite；停止输入约 1000ms 后通过串行队列自动保存。
- 可新增顶层章节或当前章节下的小节；可删除、上移和下移章节。
- 预览按钮在编辑与渲染视图间切换，阅读按钮打开对应 Reader。
- 切换章节、正文/素材或离开内部页面前等待当前串行保存；浏览器关闭时对未保存内容给出保护提示。
- `Ctrl/Cmd+S` 保存、`Ctrl/Cmd+B` 加粗、`Ctrl/Cmd+Shift+P` 切换预览。

## 页面状态

- 章节树：展开/收起、当前选中、无章节。
- 编辑区：正文/素材、编辑/预览、空内容。
- 保存：已保存、保存中、未保存、保存失败并可重试。

## 数据字段

`DocumentSummary.id/title`、`ChapterNode` 全字段、`ChapterContent` 全字段。

## Markdown 能力

多级标题、列表/任务列表、GFM 表格、引用/提示块、行内代码、带语言标识的围栏代码块、链接、图片、删除线、分割线、行内公式 `$...$` 与块级公式 `$$...$$`。Editor 和 Reader 使用同一个渲染器。格式整理先显示前后对比，应用后可撤销。

## 响应式行为

桌面为章节树加工作区；窄屏章节树变为覆盖抽屉，头部次要文字隐藏，编辑区保持可用。代码与公式在自身容器滚动。

## 暂不实现

完整编辑历史、协同编辑、完整富文本、拖拽排序、跨文档移动章节和冲突合并。
