# Alarkive V0.1 产品规格

## 产品定义

Alarkive 是单用户、轻量级的个人学习文档平台。用户在 GPT、Gemini、DeepSeek 等外部工具中生成内容，再将 Markdown 复制到 Alarkive 中整理、保存和阅读。V0.1 的闭环是 `Library → Editor → Reader`。

## 用户目标

1. 快速建立一份结构清晰的学习文档。
2. 按章节粘贴并整理 Markdown，而非重新排版。
3. 在低干扰的教材式界面中持续阅读并保留进度。

## 页面与核心任务

| 页面    | 目标               | 核心动作                            | 主要数据              |
| ------- | ------------------ | ----------------------------------- | --------------------- |
| Library | 浏览与定位个人文档 | 搜索、新建、继续阅读、进入编辑      | 文档摘要、标签、进度  |
| Editor  | 建立结构并整理内容 | 管理章节、编辑正文/素材、预览、保存 | 章节树、Markdown 内容 |
| Reader  | 连贯地阅读整本文档 | 章节跳转、调节阅读体验、记录进度    | 正文、目录、阅读进度  |

## 共通状态

- 加载：V0.1 从本地 SQLite 同步读取，界面预留骨架屏演进空间。
- 空数据：无文档、无章节、章节无内容。
- 错误：未知文档返回 404；数据库或写入失败显示明确错误，不回退示例数据。
- 响应式：桌面优先，窄屏通过抽屉或隐藏次要栏保留核心流程。

## V0.1 范围

- SQLite 与 Drizzle ORM 驱动的三页面个人工具。
- Markdown 编辑与渲染，支持 GFM、代码和 KaTeX 公式。
- 文档、章节、Markdown 与阅读进度真实持久化；阅读显示设置保存在 localStorage。
- 页面间使用稳定的文档 ID 路由跳转。

## 暂不实现

AI API、账号、多用户、社区、Explore、在线 Fork、支付、推荐、协作、云同步、数据库、附件上传、完整移动端、所见即所得编辑和拖拽排序。
