# 设计变量

## 设计原则

克制、轻量、安静。用浅灰背景建立层次，用白色表面承载内容，用低对比边框而非重阴影分区；中文正文强调舒适行高与合理行长。

## 色彩

| Token             | 浅色      | 深色      | 用途         |
| ----------------- | --------- | --------- | ------------ |
| `--background`    | `#f5f6f8` | `#111418` | 页面底色     |
| `--surface`       | `#ffffff` | `#191d23` | 卡片与正文   |
| `--surface-muted` | `#f0f2f5` | `#232831` | 次级表面     |
| `--text`          | `#18202a` | `#edf0f4` | 主文本       |
| `--text-muted`    | `#667085` | `#9ca6b5` | 辅助文本     |
| `--border`        | `#dde1e7` | `#303743` | 边框         |
| `--accent`        | `#5967d8` | `#8490ff` | 主动作与进度 |
| `--accent-soft`   | `#eef0ff` | `#252b52` | 选中背景     |

## 排版与尺寸

- UI 字体：系统无衬线中文字体栈；正文与 UI 保持一致，降低页面噪声。
- 正文：16/18/20px 三档，默认 18px；行高 1.85。
- 最大正文宽度：紧凑 680px、舒适 780px、宽 920px。
- 圆角：8px 控件、12px 卡片、16px 大容器。
- 间距以 4px 为基准，常用 8/12/16/24/32px。
- 阴影仅用于弹窗和移动抽屉，常规分区依赖边框。

## 交互

- 可交互元素提供 hover、focus-visible 与 disabled 状态。
- 焦点环使用 2px `--accent`，不通过颜色单独表达状态。
- 动效控制在 150–200ms；尊重系统的减少动效设置。
